# Web Workspace

SvelteKit workspace UI for NovelGraph AI Platform.

## Commands

Run these from the repository root:

```sh
pnpm install
pnpm dev:web
pnpm check:web
pnpm lint:web
pnpm test:web
pnpm build:web
```

## Current Scope

- Desktop-style workspace shell.
- Projects/bookshelf view.
- Project overview, import, reading, analysis, review, and settings routes.
- Local-first UI foundation for later Rust API integration.
